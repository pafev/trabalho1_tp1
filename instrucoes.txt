##comando para compilar:

g++ *.cpp -o main

- Explicação:
    "g++" é o compilador
    "*.cpp" é porque queremos compilar todos os arquivos .cpp
    "-o main" é porque queremos renomear o executável para main (favor, sempre renomear para main)

##comando para executar (após compilado):

./main

## o que fazer depois de escrevermos o código:

iremos pegar os arquivos, dar um ctrl c + ctrl v e passar para o code::blocks
aí iremos em "build" no code::blocks, para ele compilar e criar uns arquivos
referentes ao projeto e à estrutura de arquivos da ide/ferramenta
